# CharCam

Capture the moment with chars...

[![Rust](https://github.com/jlopezcur/charcam/actions/workflows/rust.yml/badge.svg)](https://github.com/jlopezcur/charcam/actions/workflows/rust.yml)

## Controls

- [Ctr + c] or [q] for quit
- [w] for change the "white mode"
